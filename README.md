# 1e lancement:
Nous utilisons un environnement venv, vous devrez donc en créer un nouveau et l'activer avec :
python -m venv venv && venv\Scripts\activate

Installer les dépendances grâce à :
pip install -r requirements.txt

Pour installer les dépendances Node.js du frontend (et s’assurer d’avoir la bonne version de Node.js) :
cd frontend && npm install

Les dépendances sont maintenant installées, vous pouvez lancer le projet avec :
start.bat
